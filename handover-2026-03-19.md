## 一、console-capture MCP Server

**位置**: `~/code/console-capture-mcp/`

**功能**: AI agent自动获取浏览器console日志，无需手动F12

**状态**: ✅ 完成，已配置到opencode

**配置文件**: `~/.config/opencode/config.json`
```json
{
  "mcp": {
    "console-capture": {
      "type": "local",
      "command": ["node", "/home/yslz/code/console-capture-mcp/dist/index.js"],
      "enabled": true
    }
  }
}
```

**工具**:
- `console-capture_get_console_logs` - 获取日志
- `console-capture_get_console_errors` - 获取错误
- `console-capture_get_active_sessions` - 获取活跃session
- `console-capture_clear_console_logs` - 清除日志

**关键发现**: 
- 使用 `new Image().src` 发送日志避免CORS问题
- 子agent有独立sessionId，需要通过 `state.metadata.sessionId` 获取
